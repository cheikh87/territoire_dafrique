Send Mail Sample - A simple demo how you can create console apps to send emails thanks to the PHPMailer class.

1) Modify sendmail.php to enter your SMTP server settings and the email properties.
2) Open the ExeOutput for PHP project file and compile.
3) Run the app

About PHPMailer: https://github.com/PHPMailer/PHPMailer
